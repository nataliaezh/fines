FINES
Автоматизация процесса уведомлений о наличии задолженности

Цели:
Создать полезное, актуальное и удобное приложение для автоматизации процесса нотификации должников о наличии задолженности.

Проблема:
Сотрудники вынуждены выполнять однообразую, трудоемкую работу в рамках документооборота компании. Процесс автоматизации позволит с помощью 
приложения Fines повысить эффективность работы и уменьшить расходы предприятия.

Решение:
Данное приложение должно предоставить пользователю возможность формировать документы в форматах html и doc для дальнейшей отправки адресату или иных целей.


Запуск проекта
Для запуска вам потребуется:

Скачать архив приложения (или выполнить fork в свой репозиторий и клонировать его на свой ПК).
На ПК полжна быть запущена база данных Postgres. Она должна работать на стандартном порту (5432).
В базе данных необходимо создать пользователя 'postgres' и установить ему пароль. Далее внести эти данные в файл application properties приложения в соотвествующие поля.
В базе необходимо создать схему 'fines', выполнить команду:
SET SEARCH_PATH to fines; 
Далее открыть файл fines_script_draft1.txt и скопировать содержимое файла в схему.
   
Проект готов к запуску.
Технологии:
